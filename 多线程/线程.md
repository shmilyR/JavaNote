# 线程  
    程序的执行线索。
+ 传统创建线程的方式：  
1、Runnable : 线程所要运行的代码的宿主。Runnable对象不是线程。new Runnable()
更能体现面向对象的思维。  
```
    Thread t1 = new Thread(new Runnable(){
        重写run()方法
    });
```
2、Thread : new Thread的子类
```
Thread t1 = new Thread(){
    重写run方法
};
```  
3、下面这种写法，会执行new Thread()下的run()方法。看源码  
```
new Thread(){
    new Runnable() {
        重写run()方法
    }
    重写run()方法。
}.start();
```
总结：多线程并不会提高程序的运行效率。
